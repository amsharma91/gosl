    func Delaunay(X, Y []float64, verbose bool) (V [][]float64, C [][]int, err error)

    Delaunay computes 2D Delaunay triangulation using Triangle 

    Input:
  X = { x0, x1, x2, ... Npoints }
  Y = { y0, y1, y2, ... Npoints }
Ouptut:
  V = { { x0, y0 }, { x0, y0 }, { x0, y0 } ... Nvertices }
  C = { { id0, id1, id2 }, { id0, id1, id2 } ... Ncellls }
